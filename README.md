# Batalla Campal Sin-igual 🎮

Proyecto II de Introducción a la Programación - Juego de estrategia por turnos en Python.

## 📋 Descripción

Un juego estratégico por turnos donde dos jugadores compiten:
- **Defensor**: Construye torres y defensas para proteger su base
- **Atacante**: Despliega tropas para destruir la base enemiga

## 🎯 Características

✅ Sistema de registro e inicio de sesión  
✅ Rankings de victorias (atacante/defensor)  
✅ 5 facciones distintas (Medieval, Futurista, Naturaleza, Oscura, Robótica)  
✅ Sprites personalizados por facción  
✅ Sistema de combate en tiempo real  
✅ Gestión de recursos (oro)  
✅ Persistencia de datos en JSON

- Python 3.8+
- Tkinter (incluído en Python)
